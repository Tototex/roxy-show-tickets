Custom ticketing and show management system for the Newport Roxy Theater.

== Description ==

Roxy Show Tickets is a custom-built ticketing and show management system designed specifically for theater operations.

Features include:

* Show scheduling
* Automatic WooCommerce ticket product creation
* Capacity tracking
* QR ticket generation
* Door mode scanning
* Membership integration
* Will-call compatibility
* Weekend duplication tools
* Legacy ticket bridge support
* Sales dashboard per showing

This plugin is part of the Newport Roxy Theater management system.

== Installation ==

1. Upload plugin zip to WordPress
2. Activate plugin
3. Configure show settings
4. Create showings
